üõ† Installation

Prerequisites

  • Flutter SDK 3.8.1+
  • Dart SDK 3.0+
  • Android SDK (API 35)
  • VS Code / Android Studio

Clone & Run

git clone https://github.com/deboo92/taskmate.git
cd TaskMate
flutter pub get
flutter run             # Debug
flutter run --release   # Release
